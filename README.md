# CS:GO Marketplace
Темный маркетплейс игровых предметов CS:GO

## Запуск через Docker (для однокурсников)

### Простая инструкция:

1. **Установите Docker Desktop**
   - Скачайте с [официального сайта](https://www.docker.com/products/docker-desktop/)
   - Установите и перезагрузите компьютер

2. **Скачайте проект**
   - На GitHub нажмите **Code → Download ZIP**
   - Распакуйте архив в любую папку

3. **Запустите проект**
   - Откройте папку с проектом
   - **Зажмите Shift + Правой кнопкой** по пустому месту
   - Выберите **"Открыть окно PowerShell здесь"**
   - Введите команду:
     ```powershell
     docker-compose up
     
4. **Откройте в браузере**
http://localhost:8080

### Остановка:
- В PowerShell нажмите `Ctrl + C`
